Библиотека для упрощенной работы с консоли.
Основано на iostream

Функции выывода в консоль:
  Существуует два "типа":
    1. Вывод без переноса строки
    2. Вывод с переносом строки (с \n в конце)
  
  Типы данных для выывода в консоль:
   1. String - строковый
   2. Integer - числовой, целое число
   3. Float - числовой, дробное число
